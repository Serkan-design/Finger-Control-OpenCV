# Hand Gesture Control System

## 📌 Project Description
This project is a real-time hand gesture control system built using MediaPipe and OpenCV.  
It detects hand landmarks via webcam and processes finger movements.

## 🚀 Technologies Used
- Python
- MediaPipe
- OpenCV

## 🎯 Features
- Real-time hand tracking
- Finger landmark detection
- Gesture recognition
- Webcam-based control

## 🖥️ Requirements
Install required libraries:

pip install -r requirements.txt

Or manually:

pip install opencv-python mediapipe

## ▶️ How to Run
1. Clone the repository
2. Install dependencies
3. Run the Python file:

python hand_gesture_control.py
